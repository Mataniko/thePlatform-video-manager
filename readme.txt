Manage your content hosted by thePlatform and embed media in WordPress posts.

== Description ==
View your content hosted by thePlatform for Media and easily embed videos from your
library in WordPress posts, modify media metadata, and upload new media. 
  
== Installation ==

Copy the folder "thePlatform-video-manager" with all included files into the "wp-content/plugins" folder of WordPress. Activate the plugin and set your MPX credentials in the plugin settings interface.

== Screenshots ==

1. Set your MPX credentials and other general plugin preferences
2. Set the metadata fields that you wish to have available for searching
3. View your media library, search for videos, and sort the returned media set
4. Modify video metadata
5. Upload media to your MPX account
6. Embed media from your library into a WordPress post

== Configuration ==

Ensure that an appropriate account exists in MPX with sufficient privileges to access your desired media and to
upload new media. Additionally, you must configure a publishing profile in MPX names "Wordpress". This publishing
profile should be capable of creating streaming media and generating appropriate thumbnail images. Refer to pertinent
MPX documentation to learn more about this.
